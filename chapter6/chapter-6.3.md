# 6.3 ไดรฟ์โซลิดสเตต (Solid State Drives - SSD)

SSD เปรียบเทียบกับ HDD
-----------------------
SSD แตกต่างจากฮาร์ดดิสก์แบบดั้งเดิม (HDD) ในหลายด้าน:

| คุณสมบัติ             | SSD                      | HDD                      |
|-----------------------|--------------------------|--------------------------|
| กลไกการทำงาน         | หน่วยความจำแฟลช          | จานแม่เหล็กหมุน          |
| ความเร็ว              | เร็วกว่ามาก               | ช้ากว่า                   |
| ความทนทานต่อการกระแทก | ทนทานสูง (ไม่มีชิ้นส่วนเคลื่อนไหว) | เปราะบาง         |
| เสียงรบกวน           | เงียบ                    | มีเสียงจากการหมุน         |
| การสิ้นเปลืองพลังงาน | น้อยกว่า                  | มากกว่า                   |
| ราคาต่อหน่วยพื้นที่   | สูงกว่า                   | ถูกกว่า                   |
| อายุการใช้งาน        | จำกัดจำนวนครั้งการเขียน   | ยาวนาน (หากไม่กระแทก)    |

โครงสร้างของ SSD
-----------------
- ใช้หน่วยความจำแฟลชชนิด NAND เป็นหลัก
- จัดโครงสร้างเป็น เพจ (Page) และบล็อก (Block)
- หน่วยการอ่านคือ เพจ (ปกติ 512 ไบต์ถึง 4 KB)
- หน่วยการลบคือ บล็อก (ประกอบด้วยเพจหลายเพจ)
- ไม่สามารถเขียนทับข้อมูลเดิมได้โดยตรง ต้องลบก่อนแล้วจึงเขียนใหม่

ประเด็นเชิงปฏิบัติ
--------------------
1. การสึกหรอของเซลล์ (Wear Leveling): แต่ละเซลล์มีจำนวนครั้งการเขียนจำกัด
   ระบบจัดการจะกระจายการเขียนให้ทั่วถึง
2. การขยายการเขียน (Write Amplification): การเขียนข้อมูลจริงอาจทำให้เกิด
   การเขียนจริงมากกว่าที่ร้องขอ
3. การเก็บกวาด (Garbage Collection): กระบวนการเคลื่อนย้ายข้อมูลที่ยังใช้อยู่
   เพื่อให้สามารถลบบล็อกที่ไม่ต้องการได้
